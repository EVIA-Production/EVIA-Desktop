This should be fixed in v1.0.2. If you still see it:
1. Open Terminal
2. Run: `xattr -cr /Applications/EVIA.app`
3. Try opening again

EVIA needs these permissions to work:
- **Microphone**: To hear you
- **Screen Recording**: To capture system audio (meeting audio)
